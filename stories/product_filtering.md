# User Story: Product Filtering by Category

**As a** customer of the dress shop  
**I want** to be able to filter products by their category  
**So that** I can quickly find the specific types of dresses I am looking for without browsing through the entire collection

## Acceptance Criteria

1. **Category Selection:**
   - The user interface must display a list of available product categories (e.g., Casual, Evening, Summer, Workwear).
   - Only one category can be active at a time (simple filter).

2. **Filtering Logic:**
   - When a category is selected, the product grid must immediately update to display only products belonging to that category.
   - If no products are available in the selected category, a "No products found" message should be displayed.

3. **Resetting Filters:**
   - There should be an "All" or "Clear Filter" option to return to the full list of products.
   - The default state should show all products.

4. **Visual Feedback:**
   - The currently selected category should be visually highlighted to indicate it is active.
